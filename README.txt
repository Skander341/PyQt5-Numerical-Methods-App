This project is a Python desktop application built using PyQt5. It provides a graphical interface to
perform numerical computations, including solving equations, approximating functions, and
calculating integrals.
Main Features:
- Computes roots of equations using iterative and bisection methods
- Approximates logarithmic function values
- Calculates integrals using Rectangle and Trapezoidal methods
- Stores results in a binary file (Surface.dat)
- Displays stored results in a table within the GUI
Requirements:
- Python 3.x
- PyQt5 library
Installation:
pip install PyQt5
How to Run:
1. Open terminal in the project folder
2. Run: python main.py
3. Ensure interface.ui is in the same directory
Notes:
- The program must be run with all project files in one folder
- If PyQt5 is missing, install it using pip

##Notice!##:Double tap the Rectangle or the Trapzoidal sections to see the results more clearly
